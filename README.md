## Project Overview

The objective of this project is to analyze stock market trends, company performance, price movements, trading volume, and financial KPIs for MAANG companies through interactive visualizations.

- Pandas

## Dataset

The project uses historical daily stock market data of:

- Apple
- Amazon
- Google
- Meta
- Netflix

Each dataset contains:

- Date
- Open
- High
- Low
- Close
- Adjusted Close
- Volume

## Python Workflow

### Data Preprocessing

- Imported datasets
- Merged all company datasets
- Converted Date column
- Removed missing values

### Feature Engineering

Created new features such as:

- Daily Return
- Price Change
- 50-Day Moving Average (MA50)
- 200-Day Moving Average (MA200)

### Exploratory Data Analysis

Performed:

- Stock Price Trend Analysis
- Daily Return Distribution
- Correlation Analysis
- Volatility Analysis
- Top Growth Days Analysis
- COVID Market Crash Analysis
